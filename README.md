### 🏗️ [SOLID](./solid/)

**Princípios SOLID para Programação Orientada a Objetos**

Fundamentos essenciais para código de qualidade e manutenível:

- **Single Responsibility** - Uma classe, uma responsabilidade
- **Open/Closed** - Aberto para extensão, fechado para modificação
- **Liskov Substitution** - Substituição de objetos sem quebrar funcionalidade
- **Interface Segregation** - Interfaces específicas e coesas
- **Dependency Inversion** - Dependa de abstrações, não de implementações
